To run the application from the command line, run 'main.py'. This will open the GUI.

The application structure has been divided in the following directories:

- data
Contains sensor data to test with.

- data_export
Contains modules for exporting data and creating windows.

- data_import
Contains modules for importing sensor data and label data.

- datastorage
Modules for accessing data from the database.

- gen
Generated files for parsing custom functions.

- gui
All modules related to the GUI. The 'designer_...' files are automatically generated by QtDesigner.

- machine_learning
Contains the classifier module.

- parse_function
Contains modules to parse custom functions.

- projects
Contains example projects.

- resources
Images used in the GUI.

- test
Contains unit tests for various modules.
